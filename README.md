Key features:
• Fully responsive design
• Dark mode / Light mode toggle
• Clean and modern UI
• Smooth layout and spacing
• Built with reusable React components
• Optimized for performance

This project demonstrates my ability to build professional, scalable, and user-friendly interfaces for educational platforms and startups.
